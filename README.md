## Vision

OpenPICU aims to become the world's most comprehensive open-source platform dedicated to Pediatric Critical Care.

Our mission is to make high-quality, evidence-based critical care knowledge freely accessible to clinicians, trainees, educators, researchers, and healthcare institutions across the globe.

---

## Why OpenPICU?

Pediatric critical care knowledge is scattered across textbooks, journals, institutional protocols, and educational websites.

OpenPICU brings everything together into one modern platform.

It is designed to serve as a comprehensive digital workspace for pediatric intensive care.

# Installation

Clone the repository

```bash
git clone https://github.com/musaibimad/open-picu.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
