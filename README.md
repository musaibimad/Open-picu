# OpenPICU

🚑 OpenPICU is an open-source platform for Pediatric Critical Care clinicians, researchers, and trainees.

## Features

### 🫁 Respiratory
- Oxygenation Index (OI)
- Oxygen Saturation Index (OSI)
- P/F Ratio
- Mechanical Power
- Driving Pressure

### 💧 Fluids & Electrolytes
- GIR Calculator
- Maintenance Fluid Calculator
- Hypertonic Saline Calculator
- Corrected Sodium
- Anion Gap

### 💊 Drug Calculators
- Emergency Drugs
- Continuous Infusions
- Inotropes

### 📊 Clinical Scores
- pSOFA
- PELOD-2
- VIS

### 📚 Evidence Hub
- PALICC-2 Guidelines
- Pediatric Sepsis
- Status Epilepticus
- Burns
- DKA

## Mission

To provide free, evidence-based pediatric critical care tools for clinicians worldwide.
